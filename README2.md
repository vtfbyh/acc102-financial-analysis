# ACC102 Financial Analysis Tool

## Project Overview
This is an interactive Streamlit web application for financial performance analysis of three home appliance companies (Midea, Gree, Haier) from 2020 to 2024. The tool visualizes ROE (Return on Equity) trends, calculates average profitability metrics, and provides business insights for investment decision-making.

## Features
- Interactive data preview and missing value check
- Average ROE calculation by company
- ROE trend comparison chart (2020-2024)
- Data-driven business insights and investment recommendations

## Tech Stack
- Python 3.9+
- Streamlit (for web deployment)
- Pandas (for data processing)
- Matplotlib (for data visualization)
